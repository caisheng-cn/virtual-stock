## 1. 功能概述

虚拟期权交易功能允许用户对白名单内的美股标的进行期权模拟交易。支持以下操作：

| 操作 | 说明 |
|------|------|
| **买入开仓 (Buy to Open)** | 支付权利金，获得期权持仓 |
| **卖出平仓 (Sell to Close)** | 卖出持仓期权，了结头寸 |
| **行权 (Exercise)** | 实值期权可提前行权，按价差现金结算 |
| **到期结算 (Auto Settlement)** | 到期实值期权自动结算 |

### 核心规则

- 标的范围：美股个股（由管理员维护白名单）
- 定价模型：基于标的价格自动生成行权价 + 简化BS模型定价
- 货币单位：权利金以标的原币计价（如美元），实际从人民币账户扣款（按固定汇率折算）
- 合约乘数：每张 = 100股
- 到期周期：每周滚动（未来每7天一个到期日）
- 行权价：当前价 ±5 档，每档间距 = 股价 × 2.5%
- 行权方式：美式（到期前可随时行权）
- 交割方式：现金结算
- 手续费：0.5‰（与A股股票交易相同）
- 资金：与股票交易共用 `user_balance` 现金账户

## 2. 初始化配置（管理员）

### 2.1 数据库迁移

首次使用前，运行迁移脚本创建 5 张期权表：

```bash
mysql -u root -p virtual_stock < backend/migration-options.sql
```

### 2.2 添加期权标的

登录后台管理 → 期权管理 → 标的白名单，搜索并添加允许交易期权的股票。

或者直接调用 API：

```bash
# 添加 AAPL 到白名单
curl -X POST /api/v1/admin/options/whitelist \
  -H "Authorization: Bearer <admin_token>" \
  -H "Content-Type: application/json" \
  -d '{"stock_code":"AAPL","market_type":3,"stock_name":"Apple Inc."}'

# 生成期权合约（系统会自动创建11个行权价 × 5个到期日 × Call/Put = 110条合约）
curl -X POST /api/v1/admin/options/contracts/generate \
  -H "Authorization: Bearer <admin_token>" \
  -H "Content-Type: application/json" \
  -d '{"stock_code":"AAPL","market_type":3}'
```

### 2.3 管理功能

| 功能 | 路径 | 说明 |
|------|------|------|
| 白名单列表 | GET `/api/v1/admin/options/whitelist` | 分页查询 |
| 添加白名单 | POST `/api/v1/admin/options/whitelist` | 添加标的 |
| 删除白名单 | DELETE `/api/v1/admin/options/whitelist/:id` | 移除标的 |
| 启停白名单 | PUT `/api/v1/admin/options/whitelist/:id/status` | 传 status=1/0 |
| 合约列表 | GET `/api/v1/admin/options/contracts` | 可筛选状态和标的 |
| 生成合约 | POST `/api/v1/admin/options/contracts/generate` | 手动触发 |
| 手动结算 | POST `/api/v1/admin/options/settlement` | 传 expiration_date |

## 3. 用户端使用指南

### 3.1 入口

首页首页 → 点击 **期权交易** 卡片，或直接访问 `/options`。

### 3.2 交易流程

**第一步：选择标的**

在顶部标的选择框中输入股票代码或名称搜索。仅管理员添加到白名单的股票才会显示。

**第二步：选择到期日**

标的选定后，系统显示所有可用到期日标签。点击切换。

**第三步：查看期权链**

中间表格展示 **Call/Put 左右对照** 的期权链：

```
行权价   Call权利金 内在价值 时间价值  Call  |  行权价   Put权利金 内在价值 时间价值  Put
294.80   1.22      0.00     1.22     Call  |  294.80   1.22      0.00     1.22     Put
302.17   1.15      0.00     1.15     Call  |  302.17   8.52      7.37     1.15     Put
...
```

- 绿色 = Call（看涨），红色 = Put（看跌）
- 平值(ATM)行权价两边的时间价值最大
- 点击任意行可选中该合约

**第四步：买入开仓**

右侧面板显示选中合约详情：

```
行权价: $294.80
类型: Call
到期日: 2026-05-20
权利金: $1.22
内在价值: $0.00
时间价值: $1.22
剩余天数: 7天
─────────────────────
张数: [10 ▲▼]
预估权利金: $1,220.00  (~¥8,540.00)
预估手续费: $4.27      (~¥29.89)
总扣款: ¥8,569.89
[买入开仓]
```

填写张数 → 确认费用 → 点击买入开仓。

**第五步：查看持仓**

交易完成后，持仓显示在页面底部表格，或者在 `/positions` 页切换「期权持仓」Tab查看。

| 字段 | 说明 |
|------|------|
| 合约代码 | 统一编码（如 `AAPL20260520C00294800`） |
| 标的/类型 | AAPL Call/Put |
| 行权价 | 期权行权价格 |
| 到期日 | 到期日期 |
| 剩余天数 | 距到期天数 |
| 张数 | 持仓数量 |
| 均价 | 平均成本（每张） |
| 现价 | 当前权利金报价 |
| 市值(CNY) | 估值，计入总资产 |
| 虚实值 | 实值/平值/虚值 标签 |
| 盈亏 | 浮动盈亏 |

**第六步：卖出平仓**

在持仓行点击「平仓」按钮 → 弹窗选择平仓张数 → 确认。

**第七步：行权（仅实值）**

仅当期权为实值状态（ITM）时，「行权」按钮可用。点击后按价差现金结算：
- Call 行权收入 = (标的价 - 行权价) × 张数 × 100
- Put 行权收入 = (行权价 - 标的价) × 张数 × 100

### 3.3 持仓页（Positions）

`/positions` 页面顶部新增 Tab 切换：

```
[股票持仓] [期权持仓]
```

两个 Tab 各自显示对应的持仓表格，字段和操作按钮针对性设计。

### 3.4 群组广场（Group）

**排行榜**：期权持仓的市值和盈亏已纳入总资产和总利润计算，自动影响排名。

**成员详情**：持仓区新增 Tab 切换：

```
股票持仓 | 期权持仓
```

**群组消息**：新增 4 种期权消息类型：

| 标签 | 颜色 | 示例 |
|------|------|------|
| 买入开仓 | 🟡 warning | 买入开仓 AAPL Call 行权价$294.80 10张 |
| 卖出平仓 | 🔴 danger | 卖出平仓 AAPL Call 5张 盈利¥2,150 |
| 行权 | 🔵 primary | 行权 AAPL Call 5张 结算¥25,750 |
| 到期结算 | ⚪ info | 到期结算 AAPL Put 2张 结算¥3,000 |

## 4. 常见问题

### Q: 为什么我看不到期权合约？

1. 确认该股票已被管理员添加到期权白名单
2. 确认已经调用过「生成合约」API
3. 刷新期权链页面（切换到期日重新加载）

### Q: 为什么行权按钮是灰色的？

只有 **实值期权 (ITM)** 才能行权。当标的价 > 行权价（Call）或标的价 < 行权价（Put）时，按钮可用。

### Q: 期权到期会怎样？

系统每天收盘后执行定时任务：
- 检查所有当日到期的合约
- 对实值持仓自动现金结算（赚取价差）
- 对虚值持仓直接作废
- 资金自动划入账户

### Q: 期权的资金从哪里扣？

与股票交易 **共用同一个现金账户**（`user_balance.cash`）。买入期权支付权利金，卖出/行权收回资金。总资产 = 现金 + 股票市值 + 期权市值。

### Q: 期权的价格怎么来的？

使用简化 Black-Scholes 模型计算：
- 内在价值 = max(标的价 - 行权价, 0) 或 max(行权价 - 标的价, 0)
- 时间价值 = 公式计算，平值期权时间价值最大
- 年化波动率：固定 30%（后续可扩展为逐标的不同配置）

### Q: 可以同时持有多张同一合约吗？

可以。多次买入同一合约会自动合并持仓，按加权平均计算成本。

### Q: 可以部分平仓吗？

可以。在平仓弹窗中输入少于持仓总数的张数即可。

## 5. 术语表

| 术语 | 英文 | 说明 |
|------|------|------|
| 权利金 | Premium | 买入期权时支付的价格（每份合约） |
| 行权价 | Strike Price | 期权约定的买卖价格 |
| 到期日 | Expiration Date | 期权有效最后一天 |
| 看涨期权 | Call | 赋予买入权利 |
| 看跌期权 | Put | 赋予卖出权利 |
| 实值 | In The Money (ITM) | 有内在价值的期权 |
| 平值 | At The Money (ATM) | 行权价接近当前价的期权 |
| 虚值 | Out The Money (OTM) | 无内在价值的期权 |
| 内在价值 | Intrinsic Value | 立即行权能获得的收益 |
| 时间价值 | Time Value | 权利金中超出内在价值的部分 |
| 合约乘数 | Contract Multiplier | 每张合约对应股数（固定100） |
| 买入开仓 | Buy to Open | 建立期权多头头寸 |
| 卖出平仓 | Sell to Close | 了结期权多头头寸 |
| 行权 | Exercise | 行使期权权利 |
| 现金结算 | Cash Settlement | 以现金差价交割，不涉及实物股票 |
