## 4. 接口调用总结

### 4.1 可用接口列表

| 市场 | 接口类型 | 接口地址 | 状态 |
|------|--------|----------|------|
| A股 | 实时行情 | `http://hq.sinajs.cn/list=sh{code}` | ✅ 可用 |
| A股 | 历史K线 | `https://quotes.sina.cn/cn/api/json_v2.php/CN_MarketDataService.getKLineData` | ✅ 可用 |
| 港股 | 实时行情 | `http://hq.sinajs.cn/list=hk{code}` | ✅ 可用 |
| 美股 | 实时/历史 | `https://query1.finance.yahoo.com/v8/finance/chart/{code}` | ✅ 可用 |

### 4.2 注意事项

1. **新浪财经API** 需要添加 `Referer: http://finance.sina.com.cn` 请求头
2. **A股代码前缀**: `sh` = 上海, `sz` = 深圳
3. **港股代码前缀**: `hk` = 香港
4. **Yahoo Finance API**: 不需要特殊认证，但有请求频率限制
5. 日K线数据最多只能获取1023条

### 4.3 代码示例 (Node.js)

```javascript
// A股实时行情
async function getAStockQuote(code) {
  const response = await fetch(`http://hq.sinajs.cn/list=sh${code}`, {
    headers: { 'Referer': 'http://finance.sina.com.cn' }
  });
  const text = await response.text();
  const data = text.split('=')[1].replace(/"|;$/g, '').split(',');
  return {
    name: data[0],
    open: parseFloat(data[1]),
    prevClose: parseFloat(data[2]),
    price: parseFloat(data[3]),
    high: parseFloat(data[4]),
    low: parseFloat(data[5]),
    volume: parseInt(data[8]),
    amount: parseFloat(data[9])
  };
}

// 港股实时行情
async function getHKStockQuote(code) {
  const response = await fetch(`http://hq.sinajs.cn/list=hk${code}`, {
    headers: { 'Referer': 'http://finance.sina.com.cn' }
  });
  // 解析逻辑类似
}

// 美股历史数据
async function getUSStockHistory(code, startTimestamp, endTimestamp) {
  const response = await fetch(
    `https://query2.finance.yahoo.com/v8/finance/chart/${code}?period1=${startTimestamp}&period2=${endTimestamp}&interval=1d`
  );
  const data = await response.json();
  return data.chart.result[0].indicators.quote[0];
}
```
